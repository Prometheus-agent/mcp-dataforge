🐳 Docker Deployment

# docker-compose.yml
services:
 orchestrator:
  image: dataforge/orchestrator:latest
  ports: ["8080:8080"]
  environment:
   AGENT_CONFIG: /config/agents.yaml
 
 agent-pipeline:
  image: dataforge/agent-pipeline:latest
  ports: ["9001:9001"] # SSE transport
  environment:
   TRANSPORT: sse
   PORT: 9001
 
 agent-dq: # ...similar

🔌 MCP Client Integration

Contoh konfigurasi di Claude Code (~/.claude/settings.json):

{
 {"mcpServers": {
  "dataforge": {
   "command": "dataforge",
   "args": ["mcp"]
  }
 }
}

Satu entry → Orchestrator manage sisanya. Client cuma perlu satu MCP server.
