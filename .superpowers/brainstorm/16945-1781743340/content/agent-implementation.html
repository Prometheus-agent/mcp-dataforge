🧩 Agent Implementation Detail

Section 5: What each agent does internally — tools & responsibilities

🔧 Pipeline Agent

MCP Tools
  • generate_pipeline — buat pipeline dari desc
  • debug_sql — explain & optimize SQL
  • run_spark — execute & monitor PySpark job
  • explain_plan — visual query plan
  • lint_pipeline — check best practices
Integrations
  • SQL: Snowflake, BigQuery, Redshift, Postgres
  • Compute: Spark, Pandas, DuckDB
  • Transform: dbt core

✅ Data Quality Agent

MCP Tools
  • profile_data — null rate, distribution, stats
  • detect_anomalies — time-series outlier detection
  • validate_rules — run custom quality rules
  • compute_metrics — freshness, completeness, accuracy
Approach
Sampling-based profiling untuk large tables. Configurable threshold. Bisa auto-generate dbt tests dari profiling result.

📐 Schema Agent

MCP Tools
  • detect_drift — compare schema across environments
  • generate_migration — auto SQL migration script
  • lint_schema — naming conventions, data types
  • lineage — column-level lineage tracing
Capabilities
Support schema contracts. Detect breaking changes before deploy. Auto-suggest column types.

⚡ Orchestration Agent

MCP Tools
  • create_dag — generate Airflow/Dagster DAG
  • manage_retry — configure retry logic
  • resolve_deps — dependency resolution & scheduling
  • backfill — manage historical backfill runs
Integrations
Airflow, Dagster, Prefect, Cron. Read existing DAGs and suggest improvements.

📚 Catalog Agent

MCP Tools
  • search — find tables, columns, pipelines
  • describe — auto-generate documentation
  • impact_analysis — "who uses this table?"
  • tag — manage data asset tags & classifications
Backends
DataHub, Amundsen, OpenMetadata, or built-in SQLite catalog.

🔍 Observability Agent

MCP Tools
  • get_pipeline_health — pipeline status & history
  • alert_summary — recent alerts & trends
  • cost_analysis — warehouse spend & optimization
  • suggest_optimizations — cost/performance tips
Backends
Prometheus, Grafana, Datadog, or cloud-native (CloudWatch, Stackdriver).
