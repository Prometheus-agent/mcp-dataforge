↓

🔑 Key Design Principles

1

Plugin-based Agent Registry

Siapa pun bisa bikin agent plugin baru. Cukup implement Agent interface, daftarkan, langsung bisa dipanggil.

2

Shared Context Bus

Agent bisa share context, passing hasil kerja ke agent lain tanpa redundancy.

3

Tool-first Execution

Bukan cuma "ngobrol" — agent benar-benar execute SQL, run pipeline, validasi data.

4

Human-in-the-Loop

Semua destructive action butuh approval. Rollback capability built-in.
