🔌 BaseAgent Interface

class BaseAgent(ABC):
 # Metadata
 name: str       # "pipeline-agent"
 capabilities: list[str]  # ["sql", "spark", "dbt", ...]

 async def handle(msg: Message) → Response
 async def validate(task: Task) → ValidationResult
 def get_tools() → list[Tool]

🔎 Agent Discovery

Dua mekanisme:

📂 Filesystem-based

Scan plugin directories, detect pyproject.toml, load on demand. Cocok untuk development lokal.

📦 Package Registry

Install via pip: pip install dataforge-agent-dq. Register via entry_points. Cocok untuk production dan sharing.

📊 Agent Resolution Flow

Router: "Need to check data quality on customers table"
↓ capability matching
Registry: Agents with "data_quality" capability → [dq-agent, pipeline-agent]
↓ scoring (confidence + load + specialization)
→ dq-agent (score: 0.92)
→ pipeline-agent (score: 0.45)
↓ optional: parallel delegation
→ dq-agent: run data profiling
→ pipeline-agent: check upstream transformations
